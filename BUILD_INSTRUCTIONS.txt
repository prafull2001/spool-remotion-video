================================================================================
SPOOL iOS APP - REMOTION VIDEO TRAILER BUILD INSTRUCTIONS
================================================================================

This document contains the complete prompt history and instructions used to
build this Remotion video project with Claude Code (Claude AI).

================================================================================
PROJECT OVERVIEW
================================================================================

- Framework: Remotion (React-based video framework)
- Output: 1080x1920 vertical video (9:16 aspect ratio for mobile/social)
- Duration: 52 seconds (1560 frames at 30fps)
- Purpose: Promotional trailer for Spool iOS app (screen time management)

================================================================================
VIDEO STRUCTURE & TIMING
================================================================================

The video follows this slide sequence:

1. INTRO (frames 0-150) - App logo and tagline animation
2. TICKER (frames 150-330) - Scrolling stats/facts
3. LOOP SLIDE (frames 330-510) - "The Doom Scroll Loop" visualization
4. BREAK SLIDE (frames 510-690) - "Break the Loop" with Spool
5. MECHANISM SLIDE (frames 690-870) - "React → Respond" psychology visual
6. SUCCESS RATE SLIDE (frames 870-1050) - "2.9x More Likely" stat with mic icon
7. STATS SLIDE (frames 1050-1245) - Key statistics display
8. REVIEWS SLIDE (frames 1245-1425) - App Store reviews with staggered cards
9. DOWNLOAD CARD (frames 1425-1560) - Final CTA with App Store badge

================================================================================
KEY DESIGN SPECIFICATIONS
================================================================================

COLOR PALETTE (src/components/colors.js):
- Cream background: #FFF8F0
- Burnt Orange (primary): #E85D04
- Charcoal (text): #2D3436
- White: #FFFFFF

TYPOGRAPHY:
- Primary font: Quicksand (Google Font)
- Weights: 500 (regular), 600 (semibold), 700 (bold), 800 (extra bold)

ANIMATIONS:
- Spring animations using Remotion's spring() function
- Typical config: { stiffness: 180-200, damping: 16-18 }
- Glow pulse effect for emphasis
- Staggered card animations with translateX/Y and rotation

BUILD & RENDER COMMANDS
================================================================================

npm install

# Start Remotion Studio (preview)
npx remotion studio

# Render final video
npx remotion render MyVideo out/video.mp4

# Render with custom output name
npx remotion render MyVideo out/v18_reviews_bigger.mp4 --overwrite

PROMPTS USED (Summary)
================================================================================

The video was built iteratively with these key instructions:

1. "Create a Remotion video for Spool iOS app - vertical 9:16 format"

2. "Add reviews section with 4 real App Store reviews, staggered card animations"
   - Cards should alternate left/right with slight rotation
   - Spring animations for entry
   - Show 4.8 star rating and mascot at top

3. "Add App Store badge CTA at the end with 'Join 600+ users' social proof"

4. "Video gets capped at 40 seconds - increase duration to show all slides"
   - Fixed by increasing durationInFrames from 1200 to 1560

5. "Reviews section is crammed - fill the screen, make everything larger"
   - Multiple iterations (V16, V17, V18)
   - Final instruction: "Scale everything up. Way, way bigger."

6. "Remove App Store button from reviews, change bottom text to plain blue"

7. "Push to GitHub with build instructions for reference"
